如何加入 NodeJS Taiwan 開源書寫計畫？
----------------------------------

1. 請先申請一組 GitHub 帳號
2. 請瀏覽本專案於 GitHub 的網頁
   https://github.com/nodejs-tw/nodejs-from-scratch
3. 您的系統必須裝有 Git 軟體
4. 本專案的 Repo 位址是
   git@github.com:nodejs-tw/nodejs-from-scratch.git
